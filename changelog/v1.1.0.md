## Highlights

- Adds `issue.search(...)` semantic issue queries with `select` field projection for event, timer, and console triggers.
- Adds `api.notify(...)` email notifications through the official ONES notification channel.
- Supports scheduled semantic queries for timer-based automation scenarios.
- Adds `issue.hierarchy()` plus hierarchy and date guards, including `isAfter`, `isBefore`, `newValue`, and `all` / `any` / `none` status predicates.
- Moves the ONEScript workbench into the system sidebar and governs access through the ONEScript Workbench Access permission.
- Improves native full-screen mode, log and output viewers, execution-pulse display, and zh-CN / ja / en localization.

## Upgrade Notes

- Grant workbench access in the app's Permission tab after upgrade.
- Open ONEScript from the left sidebar instead of the plugin configuration tab.
- Review `taskPreAction` scripts: incoming values should use `.newValue` or `changedTo(...)`.
