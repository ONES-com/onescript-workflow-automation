# ONEScript v1.0.3

## Download

Download the ONEScript plugin package from GitHub Releases:

- [`onescript-plugin.opk`](https://github.com/ONES-com/onescript-workflow-automation/releases/download/v1.0.3/onescript-plugin.opk)
